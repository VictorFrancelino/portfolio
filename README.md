# Portfolio Web

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido para exibir minhas habilidades e conquistas na área de desenvolvimento web.

## Tecnologias Utilizadas

- **HTML5**: Estruturação do conteúdo.
- **CSS3/SASS**: Estilização e design responsivo.
- **JavaScript**: Funcionalidades e interatividade.
